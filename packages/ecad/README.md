# @modelscript/ecad

ECAD integration utilities for ModelScript. This package handles parsing, interpreting, and serializing printed circuit board schematics and CAD definitions into usable ModelScript formats.

## Scripts

| Command         | Description                          |
| --------------- | ------------------------------------ |
| `npm run build` | Clean and compile TypeScript outputs |
| `npm run test`  | Run unit tests for this sub-package  |
| `npm run lint`  | Run ESLint checks                    |
