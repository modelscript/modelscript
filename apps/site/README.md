# @modelscript/site

Main website for the ModelScript project, built with modern web technologies. This app serves the documentation, landing pages, and interactive subdomains where applicable.

## Scripts

| Command         | Description                               |
| --------------- | ----------------------------------------- |
| `npm run build` | Build the site for production             |
| `npm run dev`   | Start the development server              |
| `npm run lint`  | Run lint checks on the sub-project source |
